# Broadcast And Flux

This section documents end-to-end realtime broadcasting in Holo:

- server-side broadcasting with `@holo-js/broadcast`
- client subscriptions with `@holo-js/flux`
- framework helpers with `@holo-js/flux-react`, `@holo-js/flux-vue`, and `@holo-js/flux-svelte`

Core APIs used throughout these docs:

- `defineBroadcast(...)`
- `defineChannel(...)`
- `useFlux(...)`
